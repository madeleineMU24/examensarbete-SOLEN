### SÖLEN - Frontend
*Sol-guide för uteserveringen*

---

**Beskrivning**<br>
Frontend-delen av **Sölen** utgör applikationens användargränssnitt och gör det möjligt för användaren att interagera med systemet.

Gränssnittet är utvecklat med JavaScript och React, med Node.js som utvecklingsmiljö. Frontend kommunicerar med backend 
via HTTP-anrop till applikationens REST-API.

Genom gränssnittet kan användaren söka efter uteserveringar och kontrollera om de ligger i sol vid en specifik tidpunkt, 
eller se om solen når platsen i realtid. Informationen presenteras på ett enkelt och tydligt sätt för att göra det lätt för användaren att hitta en solig plats.

---

**Teknik**<br>

- JavaScript
- React
- Node.js

---
**Starta projektet**<br>

*Krav*

- Node.js 

**Starta med att skriva:**
```
1. npm install
2. npm run dev
```

**Applikationen startar då på:**
http://localhost:5173
